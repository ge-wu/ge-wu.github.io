5794. Sum Game

Let a be the number of question marks in Alice’s substring, and b for Bob’s substring. Note we only need check x=abs(a-b) question marks, because two questions marks on the opposite sides cancel. The first observation is that Alice will always win if x is odd. The second observation is Bob can only win if 9 x abs(cnt) / 2 != abs(diff). I first used > and got a wrong test case, and that test case made me change to !=. Sorry I cannot remember exactly how I came up with this logic during the contest. I remember I was trying to check all the cases and then find a pattern something like this. Fortunately, lee215’s post can explain this logic.
